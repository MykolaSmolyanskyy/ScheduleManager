# ScheduleManager

Back-end:
1. Spring Framework
2. H2 Database
3. Maven

Front-end:
1. AngularJS
2. Bootstrap
3. http://angular-ui.github.io/ui-calendar/

# Setup in 3 steps:
1. Install Java 8+ & Maven 3+
2. Enter project root and execute mvn package
3. Execute java -jar target/dependency/jetty-runner.jar target/ROOT.war
