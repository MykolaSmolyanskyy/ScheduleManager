# ScheduleManager

Back-end:
1. Spring Framework
2. H2 Database
3. Maven

Front-end:
1. AngularJS
2. Bootstrap
3. http://angular-ui.github.io/ui-calendar/

# Setup
1. Install Java 8+
2. Install Apache Tomcat v9+
3. Install Maven
4. Enter project root and execute mvn package
5. Copy target/ROOT.war package to the <tomcat_installation>/webapps
6. Run Tomcat Server
